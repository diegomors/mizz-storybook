- Botões principais (CTA)

- Destaques e ênfases
- Ícones principais

- Com branco: 3.10:1 (AA)

## Acessibilidade

Todas as combinações de cores seguem as diretrizes WCAG 2.1:

- **AAA**: Contraste mínimo de 7:1 para texto normal
- **AA**: Contraste mínimo de 4.5:1 para texto normal
- **AA Large**: Contraste mínimo de 3:1 para texto grande
